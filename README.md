# Shopping List

A feature-rich shopping list PWA with English / Traditional Chinese support.

## Features

- **Lists** — create, rename, delete multiple shopping lists
- **Items** — add name + optional price + quantity, edit inline, delete, check off
- **Filter / Search** — real-time text filter
- **Export / Import** — download `shopping-list.json` or re-upload
- **Language** — English and Traditional Chinese (persisted)
- **Dark Mode** — auto-detects `prefers-color-scheme`, manual toggle, persisted
- **Voice Input** — Web Speech API (EN & zh-TW)
- **Shopping Mode** — hides edit/delete/filter/input, larger checkboxes, sticky exit bar
- **Collapsible Input** — collapsed by default, FAB to show
- **Mobile Responsive** — single-column layout on small screens
- **PWA** — manifest, service worker (cache-first), installable
- **No Dependencies** — all icons are inline SVGs, no frameworks or CDN scripts
- **Storage** — all data persisted in `localStorage`

## Usage

Open `index.html` in a browser or visit the [GitHub Pages site](https://codeguy0425.github.io/shoppinglist/).
